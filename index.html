Undergraduate Research Assistant

I had the opportunity to be an undergraduate research assistant from Spring 2020 - Spring 2021, researching in areas of Machine Learning and Natural Language Processing. I was also awarded the Summer Undergraduate Research Opportunity (SUROP) grant from UMKC. Using Python as the main language, I've assisted on the following two projects.


Project 1: Given a large dataset of ratings of different male and female professors, I had to process and organize the data in order to feed it to the algorithm. The goal of the algorithm was to analyze the implicit bias between male vs. female professors given their different ratings. All findings and results were extensively visualized with MatPlotLib and Seaborn.


Project 2: Using the AWA2 dataset and the ConceptNet API, I developed a fully interacitve web application using the Dash framework to create a NLP knowledge graph that visualized word embeddings and semantic role labelling.

Smart Wheelchair

During my senior year of undergrad, I assisted a team of 3 students from the Electrical Engineering department and wrote code for different microcontrollers and sensors for their Senior Design Project. The goal of the project was to build a semi-autonomous wheelchair to aid disabled and paralyzed users with ease of mobility. Users would be able to fully control the wheelchair with a small wireless controller attached to their hands or their head. The wheelchair integrated several Arduino and Raspberry Pi microcontrollers, as well as different sensors such as gyrocscope, motor drivers, and wireless radio and bluetooth communication modules. Working on this project helped spike my understanding and expand my interest in hardware design as well as software design.

CopyCode

CopyCode is an application I built that would allow developers and software programmers to collaborate. This code-sharing platform makes it easy to securely share code online and locally. This application was built in Java using the Spring Boot framework. It also integrates the H2 relational database system. The front end interface was built with HTML, CSS, and JavaScript. I also implemented REST API endpoints in order to understand the basics of HTTP requests.
